# Work Day Scheduler

## Purpose
A website that provides a daily planner based on the working hours of 8 a.m. to 5 p.m. The planner saves the hourly events in local storage for continuous access until a new event is saved.

## Built With
* HTML
* CSS
* JavaScript
* Bootstrap
* Font Awesome
* Google Fonts
